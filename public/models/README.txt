FIGHTER MODELS
==============
Drop one GLB (recommended) or FBX per fighter id:
  public/models/nitish.glb
  public/models/krish.glb
  public/models/aarush.glb
  public/models/zain.glb
  public/models/disha.glb
Animations go in public/models/<id>_anims/ — see ANIMATIONS_README.txt.
Missing model = a built-in placeholder fighter is used.
