DROP YOUR MODELS HERE
=====================
Name files to match a fighter id and they auto-load when you run `npm run dev`:

  public/models/sora.glb    (or sora.fbx)   -> replaces SORA
  public/models/rygar.glb   (or rygar.fbx)  -> replaces RYGAR
  public/models/vela.glb    (or vela.fbx)   -> replaces VELA

GLB preferred (one self-contained file). FBX works too.
No rigging or animation needed - the game poses the whole model for you.

You can also just DRAG a .glb/.fbx onto the running game window. First click a
"Model -> NAME" button on the select screen to choose which fighter to replace,
then drop the file.
