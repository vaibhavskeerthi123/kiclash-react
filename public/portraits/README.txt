CHARACTER PORTRAIT IMAGES (optional)
====================================
Drop a square PNG/JPG named after each fighter id to show their face next to
the health bar and in character select:

  public/portraits/sora.png
  public/portraits/rygar.png
  public/portraits/vela.png

If a file is missing, a colored placeholder badge with the initial is shown.
Square images (e.g. 256x256) look best.
