TITLE FONT — Emotions Condensed
===============================
Drop the font file here to use it on the title ("BUNTY'S CATHARSIS"):
  public/fonts/EmotionsCondensed.woff2   (best) or .woff / .ttf / .otf

If the file is missing, the title falls back to a similar bold condensed
font (Oswald / Anton / Arial Narrow) automatically.

Note: "Emotions Condensed" is not a free web font, so it isn't bundled.
Obtain a license/file for it and place it here, or keep the fallback.
