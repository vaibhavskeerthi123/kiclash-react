MUSIC TRACKS (placeholders — swap freely)
=========================================
Drop your own MP3s here:
  public/audio/title.mp3    -> loading + title screen
  public/audio/select.mp3   -> character select / stage / pause / results
  public/audio/battle.mp3   -> the fight

Browsers block audio until you interact with the page, so music starts on your
first click/tap/keypress. Use the 🔊 button to mute/unmute.
If a file is missing, the game runs silently (nothing breaks).
